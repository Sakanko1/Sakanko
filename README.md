# 🏠 سكنكو (Sakanko)

منصة رقمية متكاملة لطلاب جامعة أسيوط الأهلية والجامعات القريبة، تهدف لتسهيل عملية البحث عن السكن الجامعي (شباب/بنات) بدون الحاجة لوسطاء أو سماسرة. 

"دور أقل.. اسكن أسرع!"

## 🚀 المميزات
- **بحث ذكي:** إمكانية الفلترة حسب المنطقة ونوع السكن (شباب/بنات).
- **سهولة الاستخدام:** واجهة عصرية متجاوبة مع جميع الشاشات.
- **تحديثات لحظية:** عرض السكنات المتاحة وعدد الأماكن المتبقية.
- **تفاصيل دقيقة:** عرض الصور والمميزات والأسعار لكل سكن بضغطة زر.

## 🛠 التقنيات المستخدمة
- HTML5
- CSS3 (مع تصميم مخصص وعصري)
- JavaScript (منطق برمجي متطور لعرض البيانات وتصفيتها)
- FontAwesome (للأيقونات)
